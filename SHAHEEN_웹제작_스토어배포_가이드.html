Chapter 02

웹앱 제작 (HTML)

사용 기술

별도 프레임워크 없이 순수 HTML + CSS + JavaScript 로 제작하여 모든 브라우저에서 동작합니다.

💡 Claude AI를 활용하여 코드를 생성하고 반복 수정하는 방식으로 개발하였습니다.

핵심 기능 구현

① 4:3 사진 크롭

촬영된 사진을 Canvas API를 사용해 4:3 비율로 자동 크롭합니다.

// 4:3 비율로 이미지 크롭 function crop43(img){ var r = 4/3; var cols, cH, ox=0, oy=0; if(sW/sH > r){ cH=sH; cW=sH*r; ox=(sW-cW)/2; } else { cW=sW; cH=sW/r; oy=(sH-cH)/2; } // Canvas에 크롭된 이미지 그리기 ctx.drawImage(img, ox, oy, cW, cH, 0, 0, cW, cH); }

② 사진 정보 합성

Canvas API로 사진 위에 텍스트를 직접 그려서 합성합니다.

// 상단: 작업분류 ctx.fillStyle = "#FFD600"; ctx.fillText(cat, padX, padY); // 하단 2행 2열: 날짜/TAG, 구역/팀명 botRows = [ [date + " " + time, tag], [loc, leader] ];

③ 개별 파일 저장

// dataURL → Blob → 다운로드 var a = document.createElement("a"); a.href = dataUrl; a.download = "WORK_신규_20260610_0941_PT-2101.jpg"; a.click();

GitHub Pages 배포

GitHub 계정 및 저장소 생성

1
GitHub 접속 및 로그인

github.com 접속 → 계정 로그인

2
새 저장소 생성

우측 상단 + → New repository → 이름: shaheen-photo → Public 선택 → Add README 체크 → Create repository

Source: Deploy from a branch → Branch: main / (root) → Save

Chapter 04

Expo로 Android APK 빌드

사전 준비

필요 항목설치 방법확인 명령어
Node.js (LTS)nodejs.org에서 다운로드node --version
npmNode.js 설치 시 자동 포함npm --version
EAS CLInpm install -g eas-clieas --version
Expo 계정expo.dev에서 무료 가입eas login

프로젝트 생성 및 설정

① Expo 프로젝트 생성

cd Desktop npx create-expo-app@4.0.0 shaheen-photo # SDK 버전 선택: Latest (SDK 56) → Enter cd shaheen-photo

② WebView 패키지 설치

npx expo install react-native-webview

③ App.js 수정

notepad App.js # 아래 코드로 전체 교체 후 저장
import { StatusBar } from 'expo-status-bar'; import { WebView } from 'react-native-webview'; import { SafeAreaView, StyleSheet } from 'react-native'; export default function App() { return ( <SafeAreaView style={styles.container}> <StatusBar style="light" backgroundColor="#1E3A5F"/> <WebView source={{ uri: 'https://faraoh77777.github.io/shaheen-photo/' }} style={styles.webview} allowsInlineMediaPlayback={true} javaScriptEnabled={true} domStorageEnabled={true} /> </SafeAreaView> ); } const styles = StyleSheet.create({ container: { flex:1, backgroundColor:'#1E3A5F' }, webview: { flex:1 }, });

EAS 빌드

① Expo 로그인

eas login # 브라우저에서 로그인 완료

② EAS 설정 (Git 없이)

set EAS_NO_VCS=1 eas build:configure # "Would you like to automatically create an EAS project?" → Y # "Which platforms?" → All → Enter

③ APK 빌드

eas build -p android --profile preview # Android application id: com.faraoh.shaheenphoto # Generate a new Android Keystore? → yes # 빌드 완료까지 10~15분 대기
✅ 빌드 완료 시 APK 다운로드 링크가 터미널에 표시됩니다.
또는 https://expo.dev 에서 다운로드 가능합니다.
⚠️ 빌드 진행 상황 확인:
https://expo.dev/accounts/faraoh/projects/shaheen-photo/builds/
